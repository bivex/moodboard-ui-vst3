## ✅ **BPMN 2026 — Process: Develop UI from Photos**

### 🎨 0. Define Color Palette

* [ ] Go to Paletton
* [ ] Select the main color scheme
* [ ] Save HEX / RGB values
* [ ] Define primary / secondary / accent colors

---

### 🤖 1. Generate Images

* [ ] Use **Nano Banana (latest version)**
* [ ] Before generating:

  * [ ] Request background transparency relative to the UI elements
  * [ ] Verify prompt accuracy
* [ ] Generate images

---

### 🧹 2. Remove Background

* [ ] After generation, remove the background using a dedicated tool
* [ ] Check for visual artifacts
* [ ] Export as PNG with transparent background

---

### ✂️ 3. Slice Images

* [ ] Slice images into UI components
* [ ] Separate:

  * [ ] Buttons
  * [ ] Icons
  * [ ] Cards
  * [ ] Decorative elements
* [ ] Ensure pixel-perfect dimensions
* [ ] Export in required resolutions (1x / 2x / 3x)
